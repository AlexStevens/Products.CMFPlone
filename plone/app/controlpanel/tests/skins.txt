Skins control panel
===================

First some initial setup code:

    >>> stool = self.getToolByName('portal_skins')
    >>> self.loginAsManager()

Viewing the skins control panel
-------------------------------

    >>> self.browser.open('http://nohost/plone/@@skins-controlpanel.html')
    >>> self.browser.url.endswith('skins-controlpanel.html')
    True

Click the cancel button:

    >>> self.browser.getControl(name="form.actions.cancel").click()
    >>> self.browser.url.endswith('plone_control_panel')
    True

There should be still no changes:

    >>> 'Changes canceled.' in self.browser.contents
    True

Make some changes
-----------------

    >>> self.browser.open('http://nohost/plone/@@skins-controlpanel.html')
    >>> self.browser.url.endswith('skins-controlpanel.html')
    True

    >>> self.browser.getControl(name='form.theme').value = 'Plone Tableless'

Click the save button:

    >>> self.browser.getControl(name="form.actions.save").click()
    >>> self.browser.url.endswith('skins-controlpanel.html')
    True

We should be informed that something has changed:

    >>> 'Changes saved.' in self.browser.contents
    True

Make sure the changes have been applied correctly to the mailhost:

    >>> stool.default_skin
    u'Plone Tabeless'
